# FastApi_project
FastApi_project для курса цифровой кафедры 1 курс магистратура.

Проект представляет модель торговой площадки, где пользователи могут покупать товары из списка предложений, а также выставлять не продажу свои собственные лоты.

Для этого у каждого пользователя есть собственная корзина, в которую можно добавлять лоты для дальшей покупки, и список собственных товаров на сайте.

У каждого пользователя есть собственный баланс кошелька, который ему будет необходимо пополнять для покупок. Приболь от продаж поступит на аналогичный кошелек продавца, от куда он сможет вывести средства.

Так же на сайте реализована система комментариев под товарами, как и на любой тогровой площадки.

Основные эндпоинты:


/signup -- создать нового пользователя с уникальным email


/products -- список всех товаров на сайте


/products/{id} -- просмотр деталей выбранного товара и комментариев под ним \\ добавить комментарий (требует авторизации)


/products/{product_id}/{com_id} -- редактирование\удаление своего выбранного комментария под товаром (требует авторизации)


/products/to_basket/{product_id} -- добавление выбранного товара в свою корзину (требует авторизации)


/users/ -- список всех продавцов (пользователей выставивших хотя бы один товар)


/users/me -- персональные данные, возможность изменить nickname (здесь и ниже требует авторизации)


/users/me/own_products -- список ваших товаров на продужу\\ добавление нового товара


/users/me/own_basket -- список товаров в корзине


/users/me/own_basket/buy -- POST покупка товара из корзины (нужно выбрать кол-во покупаемых товаров из одного лота)


/users/me/own_basket/{product_id} -- убрать товар из корзины


/users/me/wallet -- пополнение\снятие средств с кошелька


Запуск:
Recommended Python3.11
pip install -r requirements.txt
fastapi run ./app/main
http://0.0.0.0:8000 or http://0.0.0.0:8000/docs

Работаю в лаборатории одного из интститутов РАН. В основном занимаю моделирование теплофизических свойств, а также квантово-химическими расчетами.
Персональная информация (ФИО, факульет) Автора доступна Проверяющим через штатные каналы.
